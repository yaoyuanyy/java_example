## 简单工厂模式

简单工厂模式不是 23 种里的一种，简而言之，就是有一个专门生产某个产品的类

拿代码例子来说就是：这个shape工厂专门生产shape的，根据传入的参数不同，生产出不用的shape子类

#### factory下的集中模式的区别

简单工厂模式:
```$xslt
这是一个专门生产某个产品的工厂，根据传入的参数不同，生产出这个产品不用的子类
```

工厂方法模式:
```$xslt
这是一个专门生产某个产品的工厂，他有不同的子工厂。实例化时创建不同的子工厂，从而生成不同的产品子类
```
